# Discrete Math Toolkit

A pure Java implementation of core discrete mathematics concepts and algorithms.

The goal of this project is to learn and demonstrate discrete mathematics through software implementation rather than relying on existing mathematics libraries.

## Philosophy

* Pure Java
* No frameworks
* No external mathematics libraries
* Educational implementations
* Readable code over clever code
* Step-by-step algorithm explanations where possible

## Current Features

### Number Systems

* Binary ↔ Decimal
* Octal ↔ Decimal
* Hexadecimal ↔ Decimal
* Arbitrary Base Conversion (2–36)

## Planned Features

### Number Theory

* Prime Number Checking
* Greatest Common Divisor (GCD)
* Least Common Multiple (LCM)
* Euclidean Algorithm
* Modular Arithmetic

### Sets

* Union
* Intersection
* Difference
* Symmetric Difference
* Cartesian Product
* Power Set

### Logic

* Truth Table Generation
* Propositional Logic Evaluation

### Combinatorics

* Factorials
* Permutations
* Combinations
* Binomial Coefficients

### Relations

* Reflexive Relations
* Symmetric Relations
* Antisymmetric Relations
* Transitive Relations

### Graph Theory

* Breadth First Search (BFS)
* Depth First Search (DFS)
